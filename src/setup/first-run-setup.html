Let's get you connected to FreshBooks

Step 1: FreshBooks API Credentials

How to get your credentials:
  1. Log in to FreshBooks
  2. Go to Settings → Developer Apps
  3. Click "Create New App"
  4. Set Redirect URI to: https://localhost:8080/callback
  5. Copy the Client ID and Secret
Found in FreshBooks Developer Apps
Keep this secure - it won't be shown again

Step 2: Authorize Access

A browser window will open to authorize FreshBooks access.

Please log in and approve the connection.
